```
vue-cli-service serve [options]
    Options:
      --open    服务器启动时打开浏览器
      --copy    将URL复制到服务器启动时的剪贴板 (直接到浏览器去粘贴就OK了 http://localhost:8080/)
      --mode    指定环境模式 (默认: development)
      --host    host 地址 (default: 0.0.0.0)
      --port    端口号 (default: 8080)
      --https   使用https (default: false)
```
